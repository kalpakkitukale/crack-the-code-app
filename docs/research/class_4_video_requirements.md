## Video Sourcing Checklist

### For Each Video:
- [ ] Verify NCERT alignment
- [ ] Check video quality (720p+)
- [ ] Verify appropriate language
- [ ] Confirm correct difficulty level
- [ ] Check channel reliability
- [ ] Note video duration
- [ ] Get YouTube ID
- [ ] Get thumbnail URL
- [ ] Add to appropriate JSON file

### Quality Control:
- [ ] No ads in middle of video
- [ ] Clear audio
- [ ] Good visuals/animations
- [ ] Accurate content
- [ ] Child-appropriate
- [ ] Engaging presentation
